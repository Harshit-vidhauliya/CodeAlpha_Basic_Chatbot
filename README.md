# CodeAlpha_Basic_Chatbot

This is my Basic Chatbot project developed during the CodeAlpha Python Programming Internship.

## About

This project is a simple rule-based chatbot that responds to basic user inputs such as greetings and common questions. It uses simple Python conditions and loops to interact with the user.

## Features

* Responds to basic greetings
* Answers simple questions
* Ends the conversation when the user types "bye"
* Simple and beginner-friendly
